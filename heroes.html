Heroes

Dice Throne heroes each have unique dice faces, signature abilities, and a distinct playstyle. This table lists the officially released heroes across the base sets, Marvel and X-Men licensed sets, recent Outcasts releases, and seasonal boxes. Click any hero name (on larger screens) to link to more detailed pages you can add later.


How to read the table

  • Role - A short description of the hero's primary style (damage, control, tank, etc.).
  • Strengths - What the hero excels at in a match.
  • Set - Which release the hero belongs to (Season 1, Season 2, Marvel, X-Men, Outcasts, etc.).

Tips for exploring heroes

  • Try beginner-friendly heroes first (Barbarian, Pyromancer, Gunslinger) to learn core mechanics.
